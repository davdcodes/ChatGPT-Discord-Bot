# ChatGPT-Discord-Bot
First crack at AI implementation, using ChatGPT to write a chatbot that can be interacted with via a Discord bot.

Also the first collaborative project I've worked on, with the help of "tyedwards37" as we both experimented with both GPT and Discord APIs.

Requirements:
- Discord Developer account with Bot Token (free)
- ChatGPT Developer account with User Token (paid)
